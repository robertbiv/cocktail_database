# Cocktail Database

A Next.js web application for managing and displaying cocktail recipes with an elegant interface and comprehensive recipe database.

## Overview

This project is a modern web application built with Next.js that serves as a digital cocktail recipe manager. Users can browse through various cocktail recipes, view ingredients, preparation instructions, and discover new drinks to make at home or at a bar.

## Features

- **Recipe Database**: Comprehensive collection of cocktail recipes
- **Search and Filter**: Find cocktails by name, ingredients, or type
- **Detailed Recipe Cards**: View ingredients, measurements, and preparation steps
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean and intuitive user interface
- **Fast Performance**: Optimized with Next.js for quick loading times

## Technology Stack

- **Next.js**: React framework for production-grade applications
- **JavaScript (71.2%)**: Primary programming language
- **CSS (28.8%)**: Styling and layout
- **React**: Component-based UI library
- **Next/Font**: Automatic font optimization

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm, yarn, pnpm, or bun package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/robertbiv/cocktail_database.git
cd cocktail_database
```

2. Install dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### Development Server

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

### Editing

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## Building for Production

```bash
npm run build
npm start
```
